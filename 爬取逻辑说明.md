## 双模式设计

### 模式1：总检索（使用 Selenium）

**目标页面**：`https://www.nogizaka46.com/s/n46/diary/MEMBER?ima=5900&page=N`

**为什么使用 Selenium？**

1. **页面动态加载**：
   - 直接请求 HTML 返回空页面（没有博客数据）
   - 数据通过 JavaScript 调用 API 动态渲染
   - API 本身不支持分页参数（始终返回最新100篇）

2. **解决方案**：
   - 使用 Selenium 启动 Chrome 浏览器
   - 访问页面并等待 JavaScript 执行完成
   - 解析渲染后的 HTML 提取博客数据
   - 实现真正的分页爬取

**技术实现**：

### 模式2：指定成员检索（使用 requests）

**目标页面**：`https://www.nogizaka46.com/s/n46/diary/MEMBER/list?ct=成员ID&page=N`

**特点**：
- 成员个人页面不使用 JS 动态加载
- 可以直接使用 requests 爬取
- 每页约 10 篇博客

**技术实现**：
```python
def crawl_by_member(self, member_name, max_pages=3):
```

## 数据库设计

### blogs 表（博客信息）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT PRIMARY KEY | 博客唯一ID |
| title | TEXT | 博客标题 |
| author | TEXT | 作者名（日文） |
| author_id | TEXT | 作者ID |
| publish_date | TEXT | 发布日期 |
| url | TEXT | 博客URL |
| content | TEXT | 博客内容（HTML） |
| crawled_at | TIMESTAMP | 爬取时间 |

### images 表（图片信息）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PRIMARY KEY AUTOINCREMENT | 自增ID |
| blog_id | TEXT | 所属博客ID |
| original_url | TEXT | 原始URL |
| local_path | TEXT | 本地存储路径 |
| download_status | INTEGER | 0=未下载, 1=已下载 |
| created_at | TIMESTAMP | 创建时间 |

**约束**：
- `UNIQUE(blog_id, original_url)`：防止重复记录同一图片

### 1. Selenium 分页实现

**问题**：总博客页面使用 JS 动态加载，直接请求 HTML 拿不到数据

**解决**：
```python
def crawl_blog_list(self, start_page=1, max_pages=None):
    driver = self._create_selenium_driver()
    for page in range(start_page, max_pages + 1):
        blogs = self._parse_total_blog_page(driver, page)
        # 处理博客数据...
    driver.quit()
```

### 4. 成员ID获取

**方式1**：从数据库查找（如果模式1已爬取过）
**方式2**：从 API 实时查找

```python
def _get_member_id(self, member_name):
    # 先尝试从数据库获取
    self.db.cursor.execute(
        'SELECT DISTINCT author_id FROM blogs WHERE author LIKE ?',
        (f'%{member_name}%',)
    )
    result = self.db.cursor.fetchone()
    if result:
        return result[0]
    
    # 从 API 获取成员列表
    data = self.fetch_api("list/member")
    for item in data['data']:
        if member_name in item.get('name', ''):
            return item.get('code')
```
