# 紫团博客图片提取

一个用于爬取紫团成员博客图片的工具，支持两种操作模式。

## 功能特点

- **两种操作模式**
  - 模式1：总检索 - 爬取官网所有成员的最新博客
  - 模式2：指定成员检索 - 针对特定成员的个人博客页面进行爬取

- **智能分类存储**
  - 按成员分类
  - 按日期分类
  - 按博客ID分类（模式1）
  - 按成员+日期分类（模式2）

- **多种使用方式**
  - 图形界面（GUI）- 推荐
  - 命令行界面（CLI）

## 快速开始

### 1. 使用 GUI 版本（推荐）

双击运行 `n46blog_gui.exe`：

```
┌─────────────────────────────────────────┐
│     紫团博客图片提取 v1.3              │
├─────────────────────────────────────────┤
│  [○] 模式1：总检索（爬取所有成员）     │
│  [ ] 模式2：指定成员检索               │
├─────────────────────────────────────────┤
│  爬取页数: [1]                          │
│  成员名字: [__________] (模式2时启用)   │
├─────────────────────────────────────────┤
│  [▶ 开始爬取] [⏹ 停止] [📊 统计]      │
├─────────────────────────────────────────┤
│  运行日志窗口（实时显示）               │
└─────────────────────────────────────────┘
```

**操作步骤：**
1. 选择模式（单选按钮）
2. 设置爬取页数
3. 模式2需要输入成员日文名（如："池田 瑛紗"）
4. 点击"开始爬取"
5. 等待完成

### 2. 使用命令行版本

```bash
# 模式1：总检索
n46blog.exe full --pages 2

# 模式2：指定成员检索
n46blog.exe member --name "池田 瑛紗" --pages 3

# 查看帮助
n46blog.exe --help
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `n46blog_gui.exe` | GUI版本可执行文件 |
| `n46blog.exe` | 命令行版本可执行文件 |
| `gui.py` | GUI界面源码 |
| `main.py` | 命令行入口 |
| `scraper.py` | 核心爬虫逻辑 |
| `database.py` | 数据库管理 |
| `config.py` | 配置文件 |
| `members.py` | 成员信息映射 |

## 技术特点

- **异步下载**：使用 aiohttp 实现并发图片下载
- **Selenium 支持**：处理 JavaScript 渲染的页面
- **智能重试**：网络错误自动重试机制
- **日期解析**：从博客详情页提取准确发布时间
- **断点续传**：支持中断后继续下载

## 依赖安装

```bash
pip install -r requirements.txt
```

## 从源码运行

```bash
# GUI版本
python gui.py

# 命令行版本
python main.py full --pages 2
```

## 注意事项

1. **首次运行**：会自动下载 ChromeDriver，请耐心等待
2. **网络要求**：需要联网访问紫团官网
3. **磁盘空间**：确保有足够空间存储图片
4. **频率限制**：已设置合理延迟，避免对服务器造成压力
