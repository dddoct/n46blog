# 紫团博客图片提取 - 打包说明

## 打包文件

### 1. 命令行版本
- **文件**: `dist/n46blog.exe`
- **大小**: 约 102 MB
- **运行环境**: Windows 64位
- **特点**: 通过命令行参数操作，适合批量处理和自动化

### 2. GUI图形界面版本（推荐）
- **文件**: `dist/n46blog_gui.exe`
- **大小**: 约 102 MB
- **运行环境**: Windows 64位
- **特点**: 图形界面操作，直观易用，适合普通用户

---

## GUI版本使用方法

### 1. 启动程序
双击运行 `n46blog_gui.exe`，会打开图形界面：

```

│      紫团博客图片提取    v1.3           │

│  成员名字: [__________] (模式2时启用)   │
├─────────────────────────────────────────┤
│  [▶ 开始爬取] [⏹ 停止] [📊 统计]      │
├─────────────────────────────────────────┤
│  进度条: [████████░░░░░░░░░░] 50%      │
├─────────────────────────────────────────┤

│  │ >>> 正在下载图片...             │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### 2. 操作步骤

#### 模式1：总检索（爬取所有成员）
1. 选择"模式1：总检索"
2. 设置爬取页数（建议1-5页）
3. 点击"开始爬取"
4. 等待爬取和下载完成

#### 模式2：指定成员检索
1. 选择"模式2：指定成员检索"
2. 输入成员日文名（如："池田 瑛紗"）
3. 设置爬取页数（建议3-5页）
4. 点击"开始爬取"

### 3. 功能按钮
- **▶ 开始爬取**: 开始爬取和下载
- **⏹ 停止**: 停止当前爬取
- **📊 查看统计**: 显示数据库统计信息
- **📁 打开输出目录**: 
  - 模式1时打开 `output1/`
  - 模式2时打开 `output2/`

## 命令行版本使用方法

将 `n46blog.exe` 复制到任意目录，打开命令行运行：

```bash
# 模式1：总检索
n46blog.exe full --pages 2

# 模式2：指定成员检索
n46blog.exe member --name "池田 瑛紗" --pages 3

# 查看帮助
n46blog.exe --help
```

## 注意事项

1. **首次运行**：会自动下载 ChromeDriver，需要几秒钟时间，请耐心等待
2. **杀毒软件**：部分杀毒软件可能会误报，请添加信任
3. **网络连接**：需要联网访问紫团官网
4. **磁盘空间**：请确保有足够的磁盘空间存储图片
5. **GUI版本**：双击运行，无需命令行

## 从源码运行

### 运行GUI版本
```bash
python gui.py
```

### 运行命令行版本
```bash
python main.py full --pages 2
```

---

## 从源码重新打包

### 打包GUI版本
```bash
pyinstaller n46blog_gui.spec --clean
```

### 打包命令行版本
```bash
pyinstaller n46blog.spec --clean
```

打包后的文件在 `dist/` 目录下。
